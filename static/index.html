Help With Notable Features
  • Join Button: If you have the Viramate Chrome extension, have GBF running in another browser tab/window, have enabled the Web API setting in Viramate, and have reloaded the GBF page, the Join button will send you right to the support summon selection page for that raid provided that you meet the requirements for the raid and have enough BP.
  • Click to Copy: Click on any part of a raid's row besides the Join button to copy the raid's ID for easy copying.
  • Desktop Notifications: Click on a desktop notification to copy it's id or to join the raid if you have the Viramate integration setup.
Vertical Layout Options
  • Single Row:The "Single Row" option will just keep adding new selected raids to a single row which means that you will have to scroll horizontally after a certain number.
  • Stacking:The "Stacking" option will keep everything on the same page by putting new selected raids below the first couple like a grid. PLEASE NOTE: The "Stacking" option does not work in Chrome because Chrome has a bug that they haven't fixed yet. I will update the feature when they do fix it.

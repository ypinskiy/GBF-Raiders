News - 1/17/2019 - RAID HEALTH BETA!

Added GW raids with placeholder images for Vortex Dragon till I find better ones.

Thanks to the wonderful work done by u/kaelan_, the creator of Viramate, the site now has the ability to get current health for some raids. I will start out with a beta to test how much this will tax the site servers but here is the rundown:

  • The raid health will only be shown in the Horizontal layout and the Normal info level for now
  • The server only gets the info if you have the game and my site open at the same time, no extension support yet
  • The server only gets the info when you start a turn in the raid. Sitting idle at the skill select or whatever doesn't update it
  • The server only gets the info if you are subbed to that raid in the raidfinder
  • The server only gets the info if you tweeted the raid and it has shown up in the raidfinder

Obviously this is more accurate the more people use it and it will be more helpful in raids that aren't posted a lot so that the ones with info aren't pushed to the bottom by fresh raids. Also, I added a button in the help popup to reset the site which should be the first thing you do now if there is an issue with the site, hopefully it will solve most peoples' problems caused by weird browser cache situations.

Welcome to GBFRaiders, a raid-finding website for Granblue Fantasy.

Search for a raid in the text field above and watch them appear in real-time below. Click on the raid's row to copy the raid ID to the clipboard and paste it into the raid ID field in GBF. If you have the Chrome extension Viramate, you can enable the Web API in it's settings which will let you click the Join button by each raid to jump directly to the raid!

Never Miss A Thing

Setup desktop and audio notifications to be sure not to miss a single raid even while working on something else! Clicking on a desktop notification will copy a raid ID or directly join a raid just like when using the site. You can set custom audio notifications for even more uniqueness.

See Things The Way You Want

Switch the layout mode to Vertical to have a separate column for each raid type with separate notification settings for each. Set the Info Level to control how much information about the raid is visible, either to save space or to see Twitter messages.

Open Source

The source code of this site can be found on GitHub. Review it all you want, to see how things tick or to make sure your data is safe. Fork your own copy to run a custom version for yourself or your clan.

Extensible

GBFRaiders has a companion Chrome extension: GBFRaiders 2. Think of a simplified version of the site that works even when you don't have my main site open.
